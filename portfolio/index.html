Beta Launch - May 2026

01 - Discover Jobs
Swipe to Apply
Applicants browse job listings through a swipeable card deck. Swipe right to apply, left to pass - a familiar, low-friction interaction that makes job hunting feel less like work.
02 - Two-Sided Marketplace
Built for Both Sides
Businesses post listings and move applicants through a hiring pipeline (Interested → Hired). Applicants track every application in one place, with active and archived views.
Business - Listings
Applicant - Quests
03 - Real-Time Messaging
Chat When It Clicks
In-app messaging connects businesses and applicants the moment there's mutual interest. Live updates via WebSocket, typing indicators, read receipts, and push notifications keep conversations moving.
Under the Hood
JWT + Refresh Token Rotation
Short-lived access tokens with silent refresh - users stay logged in without re-authenticating.
Apple & Google SSO
Native OAuth flows on iOS alongside email/password auth, all sharing the same session pipeline.
WebSocket Messaging
Live chat with a polling fallback - messages arrive instantly without requiring a page refresh.
Per-Route Rate Limiting
Sliding window limiter keyed by IP, with separate thresholds for auth vs. registration routes.
Stripe + RevenueCat
Stripe for business subscriptions, RevenueCat for applicant in-app purchases - two billing surfaces, one codebase.
AWS S3 Media Uploads
Profile photos and video intros upload directly to S3, keeping the API server stateless.
Built with
